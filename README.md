## PR 요청하기

### 새로운 저장소 만들기
```bash
mkdir {생성하고 싶은 폴더 이름}
cd {생성 된 폴더 이름}
git init
```
### 원격 저장소에서 불러오기
```bash
git clone {원격 저장소 주소}
```

### 원격 저장소 목록 확인
```bash
git remote -v
```
### origin 원격 저장소 추가
```bash
git remote add origin {원격 저장소 주소}
git fetch origin
```

### upstream 원격 저장소 추가
```bash
git remote add upstream {PR 보낼 원격 저장소 주소}
git fetch upstream
```
